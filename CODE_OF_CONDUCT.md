# Code of Conduct

This project adopts the **Contributor Covenant 2.1**.

Read the full text:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you observe behavior that violates the Contributor Covenant in this project's
spaces — its issues, pull requests, discussions, or anywhere else where the
project is represented — please report it via a [private security advisory](https://github.com/WikipediaBrown/napkin/security/advisories/new),
which lets you reach the maintainer privately through GitHub without exposing
your contact information publicly.

Reports are read; the maintainer aims to acknowledge within five business days.
